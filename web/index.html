About

This is a short about info of this project. For more details, check the GitHub repo of this project.

What is This?

This is a small project intended to help japanese learners. The main aim of this is to get prompts to practice grammar and vocab for making sentences, so that one can practice more "output"-ing japanese apart from the exercise that one might be doing from text/work books.

The original concept is to get some (5) words and some (3) grammar points, and try to write sentences that use them. That said you are free to use these for something else, such as on-the-spot speaking exercise! You can select upto 10 of vocab and grammar points.

You can click Get Random Selection to get randomly selected vocab and grammar points, or you can try Get Daily Selection to get random selections which change each day, similar to something like Wordle. That way you can do this in groups and check out each other's works. Note that by daily I mean each day in timezone from where you're viewing the site. I'm not inclined to work out to logic to change it uniformly across timezones ;)

How to use this

NOTE, the choices made for the levels and lessons are based on what I'm currently doing, but if you want something else, such as more jlpt levels or other book's grammar points, feel free to open an issue on the GitHub repo :)

Select one or both of the levels from the checkboxes. Set the number of words and grammar points to select, between 1 to 10. Then you can select the lesson number from Genki 1 upto (and including) which you want the grammar points from. Genki 1 is selected for the reason stated above.

After that you can select Get Random Selection or Get Daily Selection as per what you want.

You can click on the words to open their Jisho page in new tab.

There's also a dark mode switch in navbar ;)
